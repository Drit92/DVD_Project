# DVD_Project

##  Loan Applicant Visual Insights Dashboard

This repository contains an **interactive Streamlit dashboard** focused **entirely on visualization-driven insights** from loan applicant data.

The dashboard helps analyze:
- Income vs Credit relationships  
- Demographic patterns  
- Family structure  
- External stability scores  
- Cluster personas 
- Correlation insights
